Namespace: config

server. config

Manages the configurations of opengash.

Replace the strings in this file with your own values then rename it to config.js

Properties:
Name Type Description
clientID string

Google API client ID
clientSecret string

Google API client secret
redirectUrl string

Google OAuth 2.0 redirect URL.
relativeRedirectUrl string

Your redirect URL again. This time use a relative path. i.e /authenticate
hostName string

The domain name where you intend to host opengash app
cookieSignature string

A unique string to be used as a cookie signature
dbUri string

URI to your MongoDB. It can be a local DB or a cloud DB
